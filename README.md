# GRSI_Resolving_Hubble_Tension
Contains the code used in the article "Resolving the Hubble Tension with Non-Linear Relativistic Effects. A brief description of each file in the repository follows"

pymoo_GA.py - The genetic algorithm code used to simultaneously optimize the TT scalar CMB power spectrum and the Hubble tension

pymoo_plot.py - The code that generates the plots in the article

pymoo_error.py - The code that generates the uncertanties for each of the solutions presented in the article

C_L_Code - Code that evaluates the second order corrections used in the optimization and plot codes

data_(Name).txt - Data from individual papers used in the evaluation of the Hubble tension
